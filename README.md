# Angular CV Website

This folder now contains a minimal Angular 20 project structure.

## Key files and folders
- `src/index.html` — main Angular page shell
- `src/styles.css` — global styles
- `src/app/app.component.html` — main CV content template
- `src/app/app.component.ts` — component logic and data
- `src/app/app.component.css` — component-specific styling
- `src/app/app.module.ts` — Angular application module setup
- `src/environments/` — environment configuration files
